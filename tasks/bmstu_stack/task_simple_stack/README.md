# Реализация стека

## Статус тестов:
- ✅ 14 тестов пройдены
- ⚠️ 4 теста не пройдены (оптимизационные)

## Реализованные функции:
- push (lvalue и rvalue)
- pop
- top (константная и неконстантная версии)
- empty, size, clear
- emplace (perfect forwarding)
- Конструкторы: копирование, перемещение
- Операторы присваивания
- Исключения при пустом стеке

## Особенности:
- Использован std::unique_ptr для безопасности памяти
- Реализовано правило пяти (Rule of Five)
- Поддержка move semantics
